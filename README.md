# Famous Homes Nigeria

Famous Homes is a premier, high-end real estate platform localized for the Nigerian market. Part of the **Famous Holdings Group**, the platform provides a seamless, modern experience for discovering luxury residential and commercial properties in Lagos and across Nigeri

- **Dynamic Interactive Hero**: A high-impact background slideshow featuring premium Nigerian architecture.
- **Localized Content**: All prices in Naira (₦) with a focus on premium neighborhoods like Ikoyi, Banana Island, and Lekki.
- **Multi-Page Architecture**: Dedicated sections for Property Listings, Company Heritage, and Private Consultations.
- **Mobile Optimized**: Fully responsive design tailored for a premium experience on Android and iOS devices.
- **Corporate Aesthetic**: A sophisticated "Very Light Blue" theme designed to reflect trust, professionalism, and excellence.

## Tech Stack

- **React 19** with **Vite**
- **Tailwind CSS 4** for modern, high-performance styling
- **React Router 7** for seamless navigation

## Deployment

This project is configured for easy deployment on **Vercel**:

1. Push to GitHub
2. Connect the repository to Vercel
3. The build settings are automatically detected (Framework: Vite)
